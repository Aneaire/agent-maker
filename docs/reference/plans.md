## Tool Set Availability

All tool sets are available on all plans. Some tool sets require additional configuration:

| Tool Set | Requirements |
|----------|-------------|
| Email | Resend API key |
| Knowledge Base | Documents uploaded + GEMINI_API_KEY |
| Custom HTTP | Tool endpoints configured in Settings |
| REST API | API page type (Pro+) + API key generated |
| PostgreSQL | Connection string configured (Pro+) |
